A Semântica do HTML5 vs. a Forma do HTML4

O HTML 4 veio antes do HTML5: hoje, está em desuso. A recomendação do W3C, que mantém o padrão da linguagem HTML, é o uso do HTML5. Na prática, isso significa que o HTML5 é uma linguagem semântica, em detrimendo da sua versão de forma no HTML4.

Semântica e Forma

As formas são as opções de modelar o que se vê no texto escrito marcado pelas tags. Por exemplo, o negrito, o sublinhado, o itálico, a cor de fundo, cor da fonte, pequenas animações e etc. Tudo que altera visualmente o conteúdo do arquivo HTML. Antigamente, se fazia assim. Hoje isso não é mais válido.

O HTML5 veio para se tornar uma linguagem semântica: preza-se o conteúdo, e não mais o visual. O visual, no caso do HTML5, entrega-se para o CCS3, enquanto que as animações são feitas via JavaScript (JS).

Impacto

O uso correto atualmente é o HTML5. Caso você insista em utilizar tags obsoletas, os mecanismos de busca irão priorizar outros sites em detrimento do seu: o que impacta diretamente em como você "rankeia" nos resultados indexados de uma busca.

É preciso ter cuidado.
