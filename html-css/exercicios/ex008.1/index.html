Código-fonte

OS códigos de que falamos são os comandos em uma linguagem de programação. Se escritos na fonte Times New Roman, ou alguma outra que não seja mono-espaçada, ou seja, largura de letra uniforme para todos os caracteres, fica difícil sua legibilidade. Por isso, usamos tags para transformar o código-fonte em uma legibilidade melhor com uma letra mono-espaçada. Verifique o código-fonte deste documento para saber a tag usada no seguinte comando:

document.getElementById('Teste')

A legibilidade do código acima ficou mais adequada.

Para criar a identação de um código-fonte é preciso uma outra tag. Verifique as tags deste documento para saber como foi feito o alinhamento adequado do código Python a seguir:

            
    num = int(input('Digite um número: '))
    if num % 2 == 0:
        print(f'O número {num} é par.)
    else:
        print(f'O número {num} é ímpar.)
    print('Fim do programa.')

Citações Simples

As citações simples são aquelas que são na mesma linha, citações pequenas, como por exemplo a seguinte citação. Como diria o pai de um amigo: O computador é o burro mais rápido que existe.

Citações Completas

As citações completas são aquelas que, por serem maiores, deslocam um bom espaço de bloco de citação. Estas citações têm sua linha própria e normalmente ficam indentadas no navegador. Vamos a um exemplo.

Segundo Susan Strange, 1996:

They are a way of being a signpost, pointing not along an open well-trodden track but rather into a mysterious forest of the unknown. Just where the path will lead, I am not at all sure. That is the nature of exploration - and its appeal to the mentally adventurous. (STRANGE, 1996).
